Text Similarity Checker is a Java application that compares two pieces of text and determines how similar they are. The system calculates a similarity percentage and helps users identify duplicate or closely related content.

Features
Compare two text inputs
Calculate similarity percentage
Simple and user-friendly interface
Fast text analysis
Duplicate content detection
Technologies Used
Java
Object-Oriented Programming (OOP)
Java Swing (for GUI, if applicable)
How to Run
Install Java JDK 17 or later.
Open the project in VS Code or any Java IDE.
Compile the source files:
javac *.java
Run the main class:
java TextSimilarityChecker
Usage
Enter the first text.
Enter the second text.
Click the "Compare" button.
View the similarity percentage and result.
